Key behaviours
Launcher

• ⌘K or toolbar button opens it
• Type to filter chats in real-time
• ↑↓ to navigate, ↵ to open, Esc to dismiss
• Clicking outside dismisses
• Quick actions always visible at top
Floating panel result

• Selection always opens in floating panel
• If panel already open → switches to chosen chat
• New Chat → opens panel with fresh session
• Voice / Image → opens panel with mode pre-set
• Panel persists after launcher dismisses
