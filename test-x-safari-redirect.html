Server Setup

The redirect loop test requires HTTPS (since x-safari-https converts back to https). Generate a self-signed cert and serve with:

# Generate cert (once):
openssl req -x509 -newkey rsa:2048 -keyout key.pem \
  -out cert.pem -days 30 -nodes \
  -subj "/CN=localhost"

# Serve:
python3 serve_https.py

Then open https://<your-mac-ip>:8443/test-x-safari-redirect.html in DuckDuckGo iOS. You'll need to accept the self-signed certificate warning.
